When content is too large for the content box, the shorthand overflow and longhand overflow-x and overflow-y properties can be used to control what happens to the overflowing content. The overflow properties can specify that the overflowing content should be 'visible' (the default value) or completely 'hidden' from view. They can also specify that a 'scroll' bar should always be shown at the edges of the content area to let the user scroll when content overflows. The last value, 'auto', instructs the browser to only show the scroll bars if content really overflows. When the shorthand overflow property is used, it implicitly specifies both longhand properties. The overflow property is often used together with the max- width and max-height properties that specify the maximum size of the content area.
